

1. Побудуйте ланцюжок походження слів за зразком:
Несприйнятливість: (не+(((с+прийнят)+лив)+ість))
прийняти -> сприйняти -> сприйнятливий -> сприйнятливість -> несприйнятливість
атиповий: (а+((тип)+овий))
тип -> типовий -> атиповий
безвідповідально: (без+(відповід+ально))
відповідь -> відповідальний -> відповідально -> безвідповідально
мореплавання: ((мор+е)+((плав+анн)+я))
море+[плавати -> плавання]
оподаткувати: ((о+((подат+ку)+ва))+ти)
податок -> податкування -> оподаткування -> оподаткувати
перевтілитися: ((пере+(втіл+ити))+ся)
втілення -> перевтілення -> перевтілитися
схилившись: (с+((хил+ив)+шись))
хилити -> схилити -> схилившись
трьохярусний: ((тр+ьох)+(ярус+ний))
[три -> трьох]+[ярус -> ярусний]
підсніжник: ((під+сніж)+ник)
сніг -> сніжний -> підсніжний -> підсніжник
зужиткований: (з+(ужит+кован)+ий)
ужитий ->  зужитий -> зужиткований


2. Перевірте роботу [SnowballStem](http://snowballstem.org/) для спільнокореневих слів. Напишіть ваші спостереження.
import snowballstemmer
stemmer = snowballstemmer.stemmer('english')
print(stemmer.stemWords("truth truthful truthfulness countertruth untruthful truthology".split()))
print(stemmer.stemWords("flaw flaws flawed flawless flawlessness flawlessly".split()))
stemmer = snowballstemmer.stemmer('russian')
print(stemmer.stemWords("лес лесной лесник лесничий лесничество пролесье".split()))
print(stemmer.stemWords("окно окошко подоконник оконный окнище".split()))
['truth', 'truth', 'truth', 'countertruth', 'untruth', 'trutholog']
['flaw', 'flaw', 'flaw', 'flawless', 'flawless', 'flawless']
['лес', 'лесн', 'лесник', 'леснич', 'лесничеств', 'пролес']
['окн', 'окошк', 'подоконник', 'окон', 'окнищ']
З того як працює стемер можна помітити, що він практично завжди правильно визначає закінчення та плутається в префіксах (наприклад countertruth та untruthful). З суфіксами ситуація трохи краща, але можна привести приклади де вони не відділені:  flawless, flawlessness, flawlessly (тобто less залишився). З російською мовою ситуація аналогічна. В результаті можна сказати що стемер працює не дуже - в однокореневих словах, приводить до різних коренів, часто залишає префікси та суфікси.
