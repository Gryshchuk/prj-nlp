2. Перевірте роботу SnowballStem для спільнокореневих слів. Напишіть ваші спостереження.

truth, truthful, truthfulness, countertruth, untruthful, truthology
flaw, flaws, flawed, flawless, flawlessness, flawlessly,
лес, лесной, лесник, лесничий, лесничество, пролесье
окно, окошко, подоконник, оконный, окнище

SnowballStem плохо выявляет корень слова. Отрезать все приставки, суффиксы+окончания, чтобы остался только корень, он не может. Может справится только если есть один суффикс. Если в слове больше одно суффикса, он отрезает только последний. Если есть приставка - не отрезает, суффикс+окончание - убирает только окончание.

корень + суффикс - truthful, flaws, flawed, flawless, лес, лесной: правильно отрезает
корень + суффикс+ суффикс - truthfulness, flawlessness, лесник: суффксы одинаковые, отрезало только у 
приставка + корень + суффикс - countertruth, untruthful, пролесье: отрезает только суффикс, приставка остается
корень + суффикс + окончание - truthology, flawlessly, лесничий: отрезает только окончание, суффикс остается
